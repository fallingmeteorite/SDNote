# K 采样器
K 采样器会根据提供的特定的模型和正、负两种条件，来改造提供的原始潜在图像信息。 首先，它会根据设定好的随机种子和降噪强度，给原始图像数据加入一些噪声，然后输入预设的模型结合正向和负向的引导条件，去生成图像。


## 输入
|参数名称|数据类型|作用|
|---|---|---|
|模型|模型|输入用于降噪过程的模型。|
|正向条件|条件|用于引导降噪的正向条件，可理解为想要在画面中出现的内容。|
|负向条件|条件|用于引导降噪的负向条件，可理解为不想要在画面中出现的内容。|
|潜空间图像|潜空间|用于降噪的潜空间图像。|
|随机种子|整数|用于生成随机噪声，使用同样的种子可以生成相同的画面。|
|运行后操作|界面选项|提供在每次提示后更改上述种子数的能力，节点可以随机、增量、减量或保持种子数不变。|
|步数|整数|去噪过程中要使用的步骤数，允许采样器采取的步骤越多，结果就越准确。|
|CFG|浮点数|控制生成的图像多少比例上接近输入的条件，通常这个值在4~8左右。|
|采样器|界面选项|选择用来降噪的采样器。|
|调度器|界面选项|控制模型在扩散过程噪声如何进行变化。|
|降噪|浮点数|取值 0~1 之间，决定去除多少比例的噪声，在图生图中，值越小生成图像和输入图像关联越小，值越大和输入图像越像。|


## 输出
|参数名称|作用|
|---|---|
|潜空间图像|输出经过采样器降噪后的潜空间图像。|


# K 采样器(高级)
K 采样器会根据提供的特定的模型和正、负两种条件，来改造提供的原始潜在图像信息。 首先，它会根据设定好的随机种子和降噪强度，给原始图像数据加入一些噪声，然后输入预设的模型结合正向和负向的引导条件，去生成图像。


!!!tip
    鉴于结束降噪步数 >= 步数，K 采样器(高级)节点将以与K 采样器节点完全相同的方式对潜空间噪声进行降噪，降噪设置为`降噪 = (步数 - 开始降噪步数) / 步数`。


## 输入
|参数名称|数据类型|作用|
|---|---|---|
|模型|模型|输入用于降噪过程的模型。|
|正向条件|条件|用于引导降噪的正向条件，可理解为想要在画面中出现的内容。|
|负向条件|条件|用于引导降噪的负向条件，可理解为不想要在画面中出现的内容。|
|潜空间图像|潜空间|用于降噪的潜空间图像。|
|添加噪波|去噪前是否给潜在图像添加噪声。启用后，节点将注入适合给定启动步骤的噪声。|
|随机种子|整数|用于生成随机噪声，使用同样的种子可以生成相同的画面。|
|运行后操作|界面选项|提供在每次提示后更改上述种子数的能力，节点可以随机、增量、减量或保持种子数不变。|
|步数|整数|去噪过程中要使用的步骤数，允许采样器采取的步骤越多，结果就越准确。|
|CFG|浮点数|控制生成的图像多少比例上接近输入的条件，通常这个值在4~8左右。|
|采样器|界面选项|选择用来降噪的采样器。|
|调度器|界面选项|控制模型在扩散过程噪声如何进行变化。|
|开始降噪步数|界面选项|确定在计划的哪一步开始去噪过程。|
|结束降噪步数|界面选项|确定在计划的哪一步结束去噪。当此设置超过步数时将结束降噪。|
|返回噪波|界面选项|禁用后，K 采样器(高级)将尝试在最后一步中完全消除潜空间噪声。根据跳过的计划中的步骤数量，输出可能不准确且质量较低。|


## 输出
|参数名称|作用|
|---|---|
|潜空间图像|输出经过采样器降噪后的潜空间图像。|
